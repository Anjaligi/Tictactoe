# TIC TAC TOE GAME

A fully interactive **Tic Tac Toe Game** built **HTML, CSS, AND JavaScript**.
This Project provides both **SinglePlayer (vs Bot in all three easy, medium and hard mode)** and **MultiPlayer (vs Friend)** modes with advanced features like game Continuation, statistics tracking, themes and sounds for an engaging user experience.

---

## ✨ Features

1. **Game Modes**
   - 🎲 Singleplayer (play against bot in either easy, medium or hard mode).
   - 👥 Multiplayer (play with your friend on the same device (you can input names of both person also).

2. **Undo and Exit**
   - can undo (till game not ends)
   - can exit (back to home directly)
  
2. **Game Continuation**
   - can continue the game from where you left off- even after closing browser.

3. **Statistics Tracking**
   - Track **current match results**.
   - Track **lifetime records can be seen on refreshing the browser**, since the first play on device.

4. **Themes & Dark Mode**
   - Multiple themes to customize your game board appearance.
   - Toggle between **Light / Dark mode** with a single click

5. **Responsive & Engaging UI**
   - Clean, attractive, and mobile-friendly design also.
   - Background music and sound effects for:
       - Cell clicks.
       - Win/ Lose/ Draw events
